Detailed Description

The Lazy implementation of Prim algorithm class.

This class is used for generating an MST using Prim's algorithm This implementation of Primâ€™s algorithm uses a priority queue to hold crossing edges, a vertex-in-dexed arrays to mark tree vertices, and a queue to hold MST edges. This implementation is a lazy approach where we leave ineligible edges in the priority queue.
